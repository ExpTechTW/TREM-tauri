## 關於 TREM
TREM 是一款開源地震速報軟體，提供給您即時的地震資訊，利用自製的測站，顯示各地的即時震度，在地震發生的第一時間取得各管道發布的強震即時警報訊息

## 強震即時警報來源
* [交通部中央氣象署](https://www.cwa.gov.tw/)
